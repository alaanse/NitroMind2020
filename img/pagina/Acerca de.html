ACERCA DE

Un sistema de numeración es un conjunto de números que poseen ciertas reglas y convenios para nombrar y escribir los números naturales. Entre ellos podemos encontrar, los números binarios, octales, decimales y hexadecimales.

1. Decimales.

Este sistema de numeración es el más usado habitualmente, su base es de 10 y contiene las cifras 0, 1, 2, 3, 4, 5, 6, 7, 8, y 9.

2. Binarios.

Los números binarios tienen una base de 2, eso quiere decir que utilizan 2 cifras, las cuales son el 0 y el 1.

3. Octal.

Al igual que los dos anteriores sistemas de numeración anteriores, el sistema de numeración octal requiere de 8 cifras para representar cualquier número. Esta conformado por los dígitos 0, 1, 2, 3, 4, 5, 6, y 7.
